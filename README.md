# Pune Weather Pattern Analysis

Intern ID:-CITS3397

A beginner data analytics project analysing 73 years of daily weather data from Pune, India using Google Sheets.

---

## About the Project

- **Dataset:** pune.csv — 26,803 daily weather records from 1951 to 2024
- **City:** Pune, Maharashtra, India
- **Tool used:** Google Sheets (no coding)
- **Project:** 2nd data analytics project

---

## Dataset Columns

| Column | Description |
|---|---|
| Date | Date of the record (DD-MM-YYYY) |
| Rain | Rainfall in mm |
| Temp Max | Maximum temperature in °C |
| Temp Min | Minimum temperature in °C |

---

## Charts Made

1. **Monthly Average Temperature** — Line chart showing hottest and coolest months
2. **Monthly Rainfall** — Bar chart showing the monsoon pattern
3. **Yearly Temperature Trend** — Combo chart from 1951 to 2024
4. **Heatmap** — Conditional formatting colour scale on the summary table

---

## New Skills Learned

- AVERAGEIFS formula
- Combo chart with dual axis
- Conditional formatting colour scale
- Sparklines inside cells

---

## Key Findings

- April and May are the hottest months in Pune (~35°C)
- July has the highest rainfall — peak of the monsoon season
- December and January are the driest and coolest months
